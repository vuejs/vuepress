# @vuepress/plugin-blog

> Blog plugin

## Install

```bash
yarn add -D @vuepress/plugin-blog
# OR npm install -D @vuepress/plugin-blog
```

## Usage

```javascript
module.exports = {
  plugins: ['@vuepress/blog'] 
}
```

## Options

### categoryIndexPageUrl

- Type: `string`
- Default: `/category/`

### tagIndexPageUrl

- Type: `string`
- Default: `/tag/`
