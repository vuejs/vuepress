# @vuepress/plugin-google-analytics

> Google analytics 插件

## 安装

```bash
yarn add -D @vuepress/plugin-google-analytics
# OR npm install -D @vuepress/plugin-google-analytics
```

## 使用

```javascript
module.exports = {
  plugins: ['@vuepress/google-analytics'] 
}
```

## 选项

### ga

- 类型: `string`
- 默认值: `undefined`

Google Analytics ID。
