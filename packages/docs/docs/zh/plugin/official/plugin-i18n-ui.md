# @vuepress/plugin-i18n-ui

> i18n-ui 插件。

## 安装

```bash
yarn add -D @vuepress/plugin-i18n-ui
# OR npm install -D @vuepress/plugin-i18n-ui
```

## 使用

```javascript
module.exports = {
  plugins: ['@vuepress/i18n-ui'] 
}
```

## 选项

### route

- 类型: `string`
- 默认值: `/i18n/`

i18n ui页面的路径。
