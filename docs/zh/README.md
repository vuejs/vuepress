### 像 1, 2, 3 一样容易

``` bash
# 安装
npm install -g vuepress

# 新建一个 markdown 文件
echo "# Hello VuePress!" > README.md

# 开始写作
vuepress dev .

# build 静态文件
vuepress build .
```
