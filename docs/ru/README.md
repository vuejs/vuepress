### Просто как 1, 2, 3

``` bash
# устанавливаем
yarn global add vuepress # OR npm install -g vuepress

# создаём markdown-файл
echo '# Hello VuePress' > README.md

# начинаем писать
vuepress dev

# собираем в статичные файлы
vuepress build
```

::: warning ПРИМЕЧАНИЕ СОВМЕСТИМОСТИ
VuePress для работы требуется Node.js >= 8.
:::
